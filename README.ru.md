# CV Backend Service

Backend-сервис для приложения CV (Curriculum Vitae), построенный на ASP.NET Core.

Предназначен для хранения и обработки данных профилей разработчиков, а также генерации резюме и предоставления публичного API для фронтенда.

---

## 🛠 Технологический стек

- **Платформа:** .NET 10 (ASP.NET Core)
- **Язык:** C# 14
- **Базы данных:**
    - PostgreSQL
    - MongoDB
- **Хранилище файлов:** MinIO (S3-compatible)
- **Аутентификация:** JWT
- **External login:** Google OAuth
- **Генерация PDF:** Playwright (Chromium)
- **Контейнеризация:** Docker / Docker Compose
- **Логирование:** Structured Logging (Serilog)

---

## 🧱 Архитектура

Проект реализован с использованием принципов:

- Clean Architecture
- разделения ответственности
- явных границ между слоями

Основные слои:

- **Domain** — бизнес-модель и правила
- **Application** — use cases, DTO, валидация
- **Infrastructure** — БД, файловые хранилища, PDF, авторизация
- **Presentation (Web API)** — HTTP API, middleware

## ⚙️ Конфигурация

Приложение настраивается через переменные окружения.

Основные группы параметров:
- подключения к PostgreSQL и MongoDB
- JWT (issuer, audience, ключи, время жизни токенов)
- Google OAuth (client id, secret, redirect URLs)
- MinIO / S3 (endpoint, credentials, buckets)
- Playwright / PDF (параметры запуска Chromium)
- параметры окружения и публичных URL

Пример конфигурации приведён в `.env.example`.
